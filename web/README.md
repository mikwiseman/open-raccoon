# Open Raccoon Web

Minimal Next.js (App Router + TypeScript) scaffold for frontend parity work.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
npm run start
npm run test:e2e
```

## Environment

Optional variables:

- `NEXT_PUBLIC_API_BASE_URL` for REST requests
- `NEXT_PUBLIC_WS_URL` for Phoenix socket endpoint
